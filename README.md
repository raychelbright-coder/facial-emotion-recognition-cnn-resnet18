# Facial Emotion Recognition using CNN and ResNet18

Deep learning project comparing a custom CNN and ResNet18 transfer learning model for facial emotion recognition using the CK+ and JAFFE datasets.

## Technologies Used

- Python
- TensorFlow / Keras
- PyTorch
- OpenCV
- NumPy
- Google Colab

## Features

- Face detection using Haar Cascade classifiers
- Custom CNN architecture
- ResNet18 transfer learning
- Emotion classification
- Confusion matrix analysis
- Cross-dataset evaluation

## Datasets

- CK+ Dataset
- JAFFE Dataset

## Results

- Custom CNN achieved 78.35% accuracy on CK+
- ResNet18 achieved 78.18% accuracy on JAFFE
- Transfer learning demonstrated stronger generalisation across datasets

## Key Learning Outcomes

- Deep learning model development
- Transfer learning techniques
- Computer vision pipelines
- Model evaluation and performance analysis

## Academic Context

Developed as part of an Artificial Intelligence module at university.
